# man-mat-station-analysis
This project analyzes sales data from a music box. The dataset includes over 2,800 POS transaction records from the KiotViet system for 2 months. The analysis focuses on identifying patterns to find the business insights and improving revenue performance. Tools used: Excel and Power BI.
Note: The Power BI (.pbix) file and full excel file is not included because it contains the full dataset extracted from the KiotViet POS system. A dashboard preview is provided instead.
